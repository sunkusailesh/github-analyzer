# GitHub Analyzer 🚀

A simple web app that analyzes GitHub profiles and gives a score, level, and insights.

## 🔥 Features
- Analyze GitHub username
- Score calculation
- Developer level classification:
  - Beginner
  - Intermediate
  - Pro Developer
- Simple and clean UI

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS

## 🚀 How to Run
```bash
pip install -r requirements.txt
python app.py
